# Task 3 - ATM Interface

## Description
A Java console-based ATM Interface that allows users to deposit money, withdraw money, and check their account balance.

## Features
- Deposit Money
- Withdraw Money
- Check Balance
- Input Validation for Insufficient Balance
- Menu Driven Interface

## Technologies Used
- Java
- Scanner Class
- Object-Oriented Programming (Classes & Objects)

## How to Run

Compile

```bash
javac ATMInterface.java
```

Run

```bash
java ATMInterface
```
