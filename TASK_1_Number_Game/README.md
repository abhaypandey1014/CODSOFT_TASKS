# Task 1 - Number Guessing Game

## Description
This is a Java console-based Number Guessing Game developed as part of the CodSoft Java Programming Internship.

The computer generates a random number between 1 and 100, and the player has to guess it within a limited number of attempts.

## Features
- Random number generation
- User input using Scanner
- Feedback for high and low guesses
- Maximum attempt limit
- Multiple rounds
- Score calculation
- Final score display

## Technologies Used
- Java
- Scanner Class
- Random Class

## How to Run

Compile:
```bash
javac NumberGame.java
```

Run:
```bash
java NumberGame
```
